# 🎮 Clock Town Assault

## 📌 Descripción del juego
**Clock Town Assault** es un videojuego 3D en primera persona desarrollado con **Three.js**, en el que el jugador debe sobrevivir a múltiples oleadas de enemigos dentro de un entorno urbano tipo construcción.

El objetivo principal es eliminar a todos los enemigos de cada oleada mientras evitas sus ataques, utilizando movimiento, salto y combate.

---

## 🎯 Objetivo
Sobrevivir a **3 oleadas de enemigos**, eliminándolos antes de que acaben con tu vida.

## ⚙️ ¿Cómo funciona?

El juego inicia con una pantalla de bienvenida donde se muestran las instrucciones.  
Al presionar **"Iniciar"**, se activa el modo de juego:

1. Se carga el escenario 3D.
2. Se activa la cámara en primera persona.
3. Se generan enemigos en el mapa.
4. Comienza la primera oleada.
5. El jugador debe eliminar a todos los enemigos.
6. Al completar una oleada, inicia la siguiente.
7. Si el jugador pierde toda su vida → **Game Over**.
8. Si supera todas las oleadas → **Victoria**.

Los enemigos:
- Detectan al jugador
- Se mueven hacia él
- Atacan con proyectiles o ataques directos
